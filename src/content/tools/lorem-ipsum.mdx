## What is Lorem Ipsum?

Lorem Ipsum is placeholder text used in publishing and graphic design. It allows designers to focus on visual elements without being distracted by readable content.

## How to use this tool

1. Select the output type: **Paragraphs**, **Sentences**, or **Words**
2. Set the count (1-100)
3. Toggle "Start with Lorem ipsum" if you want the classic opening
4. Click **Generate**
5. Copy the result

## Why use placeholder text?

**Design mockups** — Fill layouts with realistic-looking text without writing actual content.

**Typography testing** — See how fonts render across different text lengths.

**Layout testing** — Test how your design handles various amounts of content.

**Client presentations** — Show designs without finalized copy.

## Privacy

All text is generated in your browser. Nothing is sent to any server.
