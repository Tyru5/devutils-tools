## How to use this tool

1. Select **HTML** or **XML** mode
2. Paste your code
3. Click **Format** to beautify or **Minify** to compress
4. Copy the result

## Formatting options

**Indent size** — Choose 2 or 4 spaces for indentation.

**Wrap at** — Set line length for wrapping (80, 120, or no wrap).

## HTML vs XML mode

| Feature              | HTML Mode                   | XML Mode             |
| -------------------- | --------------------------- | -------------------- |
| Self-closing tags    | Optional                    | Required             |
| Case sensitivity     | Case-insensitive            | Case-sensitive       |
| Unformatted elements | Preserves `<pre>`, `<code>` | Formats all elements |

## What gets formatted

- Proper indentation for nested elements
- Line breaks between block elements
- Attribute alignment
- Consistent spacing

## What minify does

- Removes comments
- Collapses whitespace
- Removes unnecessary line breaks
- Reduces file size

## Privacy

All formatting happens in your browser. Your code never leaves your device.
