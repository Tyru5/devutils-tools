## How to use this tool

1. Set your desired password length (4-128 characters)
2. Choose character types to include
3. Set the count if you need multiple passwords
4. Click **Generate**
5. Copy your password(s)

## Security features

**Cryptographically secure** — Uses the Web Crypto API (`crypto.getRandomValues`) for true randomness, not pseudo-random number generation.

**Strength meter** — Visual indicator showing password strength based on length and character variety.

**Client-side only** — Passwords are generated in your browser and never transmitted anywhere.

## Character sets

| Type      | Characters                    |
| --------- | ----------------------------- |
| Lowercase | a-z (26 characters)           |
| Uppercase | A-Z (26 characters)           |
| Numbers   | 0-9 (10 characters)           |
| Symbols   | `!@#$%^&*()_+-=[]{}` and more |

## Password strength tips

- Use at least 12 characters
- Include all character types
- Avoid dictionary words
- Use unique passwords for each account
- Consider a password manager

## Privacy

All passwords are generated locally in your browser. Nothing is sent to any server.
