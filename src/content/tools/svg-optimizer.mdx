## How to use this tool

1. Paste your SVG code
2. Click **Optimize**
3. See the file size reduction
4. Copy or download the result

## What gets optimized

- Removes editor metadata (Illustrator, Inkscape, Sketch)
- Removes comments and unnecessary whitespace
- Cleans up redundant attributes
- Optimizes path data
- Merges compatible elements
- Removes empty elements

## Options

**Remove comments** — Strip HTML/XML comments from SVG.

**Remove metadata** — Remove editor-specific metadata.

**Prettify output** — Format with indentation (larger file size but readable).

## Typical savings

| SVG Source         | Typical Reduction |
| ------------------ | ----------------- |
| Illustrator export | 30-60%            |
| Inkscape export    | 40-70%            |
| Hand-coded SVG     | 10-30%            |
| Already optimized  | 0-5%              |

## When to use

- Before adding SVGs to websites
- Reducing bundle size
- Preparing icons for production
- Cleaning up exported graphics

## Privacy

All optimization happens in your browser using SVGO. Your files never leave your device.
