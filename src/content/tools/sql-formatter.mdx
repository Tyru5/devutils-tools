## How to use this tool

1. Select your SQL dialect
2. Paste your SQL query
3. Click **Format** to beautify or **Minify** to compress
4. Copy the result

## Supported dialects

| Dialect      | Description                |
| ------------ | -------------------------- |
| Standard SQL | Generic SQL formatting     |
| PostgreSQL   | PostgreSQL-specific syntax |
| MySQL        | MySQL-specific syntax      |
| MariaDB      | MariaDB-specific syntax    |
| SQLite       | SQLite-specific syntax     |
| SQL Server   | T-SQL / SQL Server syntax  |
| BigQuery     | Google BigQuery syntax     |
| Redshift     | Amazon Redshift syntax     |

## Formatting options

**Indent size** — Choose 2 or 4 spaces for indentation.

**Uppercase keywords** — Convert SQL keywords (SELECT, FROM, WHERE) to uppercase.

## What gets formatted

- Proper indentation for nested queries
- Line breaks after clauses (SELECT, FROM, WHERE, etc.)
- Aligned column lists
- Formatted JOIN statements
- Consistent spacing around operators

## Privacy

All formatting happens in your browser. Your SQL queries never leave your device.
