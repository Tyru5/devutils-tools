## Features

### To Base64

Convert images to Base64 data URIs for embedding in HTML, CSS, or JSON.

### Compress

Reduce file size while maintaining quality. Great for web optimization.

### Resize

Change image dimensions with optional aspect ratio lock.

### Convert

Convert between PNG, JPEG, and WebP formats.

## How to use

1. Select a tab for the operation you need
2. Drop or upload an image
3. Adjust options if needed
4. Click **Process**
5. Download or copy the result

## Quality settings

The quality slider (10-100%) affects:

- JPEG: Compression level
- WebP: Compression level
- PNG: Not affected (lossless)

## Format comparison

| Format | Best for              | Transparency | File size |
| ------ | --------------------- | ------------ | --------- |
| JPEG   | Photos                | No           | Small     |
| PNG    | Graphics, screenshots | Yes          | Large     |
| WebP   | Both                  | Yes          | Smallest  |

## Maximum file size

Files up to 10MB are supported. Larger files may cause browser performance issues.

## Privacy

All processing happens locally in your browser. Your images never leave your device.
