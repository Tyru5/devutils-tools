## What is this tool?

The List Sorter & Formatter helps you quickly transform lists of items. Whether you have a comma-separated list, items on separate lines, or any other format, this tool can reformat and sort them in seconds.

## Input Delimiters

The tool can parse lists separated by:

- **Auto-detect** — Automatically determines the delimiter (newline, tab, comma, pipe, or space)
- **Comma** — Items separated by commas
- **New line** — One item per line
- **Space** — Items separated by spaces
- **Tab** — Tab-separated values
- **Custom** — Any delimiter you specify

## Output Formats

Choose how you want your formatted list:

- **New line** — One item per line (great for copying into code or documents)
- **Comma** — Comma-separated values
- **Space** — Space-separated
- **Tab** — Tab-separated (useful for spreadsheets)
- **Pipe** — Pipe-separated (common in documentation)

## Sorting Options

- **No sorting** — Keep original order
- **A → Z** — Alphabetical ascending (case-insensitive)
- **Z → A** — Alphabetical descending
- **0 → 9** — Numerical ascending
- **9 → 0** — Numerical descending

## Processing Options

- **Trim whitespace** — Remove leading and trailing spaces from each item
- **Remove empty** — Filter out blank entries
- **Remove duplicates** — Keep only unique items

## Use Cases

- Converting CSV data to line-separated format
- Alphabetizing a list of names or items
- Cleaning up copied data with extra whitespace
- Deduplicating lists
- Reformatting data for different applications
