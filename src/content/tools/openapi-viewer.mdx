## How to use this tool

1. Paste your OpenAPI/Swagger spec (YAML or JSON)
2. Click **Parse**
3. Explore the API structure
4. Click endpoints to expand details

## Supported formats

| Format  | Versions     |
| ------- | ------------ |
| OpenAPI | 3.0.x, 3.1.x |
| Swagger | 2.0          |

Both YAML and JSON input are supported.

## What you can see

**API Info** — Title, version, description, and servers.

**Endpoints** — All paths with their HTTP methods.

**Method details** — Summary, tags, and operation info.

## HTTP Methods

| Method | Color  | Purpose       |
| ------ | ------ | ------------- |
| GET    | Blue   | Retrieve data |
| POST   | Green  | Create data   |
| PUT    | Yellow | Replace data  |
| PATCH  | Orange | Update data   |
| DELETE | Red    | Remove data   |

## Common use cases

- Quickly preview API specs
- Review API structure before implementation
- Validate spec syntax
- Explore third-party APIs

## Privacy

All parsing happens in your browser. Your API specifications never leave your device.
