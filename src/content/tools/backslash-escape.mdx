## What is string escaping?

String escaping converts special characters (like newlines and tabs) into their backslash-prefixed representations, making them safe to use in code strings.

## How to use this tool

1. Select **Escape** or **Unescape** mode
2. Paste your text
3. Click the action button
4. Use **Swap** to switch input and output

## Supported escape sequences

| Sequence | Character | Description               |
| -------- | --------- | ------------------------- |
| `\n`     | Newline   | Line feed                 |
| `\t`     | Tab       | Horizontal tab            |
| `\r`     | Return    | Carriage return           |
| `\\`     | Backslash | Literal backslash         |
| `\"`     | "         | Double quote              |
| `\'`     | '         | Single quote              |
| `\0`     | NUL       | Null character            |
| `\b`     | BS        | Backspace                 |
| `\f`     | FF        | Form feed                 |
| `\v`     | VT        | Vertical tab              |
| `\xNN`   | —         | Hex escape (2 digits)     |
| `\uNNNN` | —         | Unicode escape (4 digits) |

## Common use cases

**JSON strings** — Prepare text for JSON that contains newlines or quotes.

**Code generation** — Create string literals programmatically.

**Debugging** — See hidden characters in text.

**Configuration** — Escape special characters in config files.
